# 생성형 AI 기반 뉴스 감성지수를 활용한 부동산 가격 예측 모델

## 프로젝트 개요

본 프로젝트는 부동산 가격 예측의 한계를 보완하기 위해 뉴스 기사 데이터를 활용한 감성 분석을 도입하고,  
경제 지표와 결합하여 아파트 매매가격지수를 예측하는 것을 목표로 합니다.

기존의 금리, GDP, 거래량 등 정형 데이터만으로는 시장의 심리적 요인을 반영하기 어렵다는 한계를 개선하고자,  
뉴스 기반 심리지수를 추가하여 예측 성능 향상을 시도하였습니다.

## 분석 과정

### 1. 데이터 수집
- 네이버 뉴스 기사 크롤링을 통해 부동산 관련 텍스트 데이터 확보
- 경제 지표 데이터를 동일 기간 기준으로 수집하여 시계열 데이터 구축

### 2. 뉴스 데이터 처리 및 심리지수 생성
- Gensim(TextRank) 및 KoBART 모델을 활용하여 뉴스 요약 수행
- ChatGPT 기반 감성 분석을 통해 뉴스 기사에 대해 긍정(1), 중립(0), 부정(-1) 점수 부여
- 월 단위 감성 점수를 집계하여 뉴스 심리지수(Sentiment Index) 생성

### 3. 데이터 전처리 및 통합
- 뉴스 심리지수와 경제 지표 데이터를 병합하여 통합 데이터셋 구성
- 금리를 제외한 변수에 대해 Min-Max 정규화 적용
- 시계열 예측을 위한 데이터 구조로 재구성

### 4. LSTM + Multi-Head Attention 모델 구축
- 시계열 예측을 위한 LSTM 기반 모델 설계
- Multi-Head Attention을 적용하여 중요 시점 정보 반영
- Adam optimizer (learning rate = 1e-3)로 모델 학습
- RMSE를 기준으로 모델 성능 평가

### 5. VAR 시계열 분석
- 경제 지표 간 상호 영향 관계 분석
- 부동산 가격과 주요 변수 간 관계 해석

## 역할 및 기여

본 프로젝트에서 데이터 수집 및 전처리 파트를 중심으로 참여하였습니다.

* 부동산 관련 키워드(아파트, 아파트 매매, 부동산 등)를 선정하여 뉴스 및 카페 데이터 크롤링 수행
* 뉴스 및 카페 데이터를 기반으로 감성 분석 데이터 구축에 기여
* LDA 기반 감성 분석을 참고하여 긍·부정 라벨링 작업 수행 (약 300건)
* GPT API를 활용한 감성 점수 산출 과정 참여
* 경제 지표 및 크롤링 데이터 결합을 위한 데이터 전처리 수행
* 팀 프로젝트 일정 관리 및 작업 분배 등 협업 과정에도 참여

또한, 팀원들과 함께 다양한 모델(ARIMA, LSTM 등)을 비교하며
모델 성능 개선 방향을 논의하였습니다.
