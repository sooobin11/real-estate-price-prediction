# 생성형 AI 기반 뉴스 감성지수를 활용한 부동산 가격 예측 모델

## 프로젝트 개요

본 프로젝트는 뉴스 데이터 기반 감성 분석과 시계열 데이터를 결합하여
부동산 가격을 예측하는 것을 목표로 수행되었습니다.

뉴스 데이터에서 추출한 감성 지표와 경제 지표(금리, GDP 등)를 활용하여
LSTM + Attention 모델과 VAR 분석을 통해 가격 변동을 분석하였습니다.

## 데이터 구성

* 뉴스 데이터 크롤링 및 감성 분석 결과
* 아파트 매매가격지수
* 매매 거래량
* 예금은행 대출금리
* GDP(국내총생산)
* 시계열 통합 데이터

※ 데이터는 프로젝트 실행이 가능하도록 일부 전처리된 형태로 포함되어 있습니다.

---

## 분석 과정

### 1. 뉴스 크롤링

* 네이버 뉴스 데이터를 수집
* 날짜별 뉴스 데이터 구축

### 2. 데이터 전처리 및 감성 분석

* 텍스트 정제 및 요약
* 감성 점수 생성
* 경제 지표 데이터와 병합

### 3. LSTM + Attention 모델

* 시계열 예측 모델 구축
* Attention Layer를 적용하여 중요 시점 반영
* 학습된 모델을 joblib 형식으로 저장

### 4. VAR 시계열 분석

* 경제 지표 간 상호 영향 분석
* 부동산 가격과 변수 간 관계 분석

## 역할 및 기여

본 프로젝트에서 데이터 수집 및 전처리 파트를 중심으로 참여하였습니다.

* 부동산 관련 키워드(아파트, 아파트 매매, 부동산 등)를 선정하여 뉴스 및 카페 데이터 크롤링 수행
* 뉴스 및 카페 데이터를 기반으로 감성 분석 데이터 구축에 기여
* LDA 기반 감성 분석을 참고하여 긍·부정 라벨링 작업 수행 (약 300건)
* GPT API를 활용한 감성 점수 산출 과정 참여
* 경제 지표 및 크롤링 데이터 결합을 위한 데이터 전처리 수행
* 팀 프로젝트 일정 관리 및 작업 분배 등 협업 과정에도 참여

또한, 팀원들과 함께 다양한 모델(ARIMA, LSTM 등)을 비교하며
모델 성능 개선 방향을 논의하였습니다.

## 느낀 점

프로젝트를 통해 데이터의 종류와 범위가 예측 결과에 큰 영향을 미친다는 점을 경험하였습니다.
뉴스 데이터뿐만 아니라 부동산 카페, 유튜브 등 다양한 사용자 기반 데이터를 활용하면 보다 현실적인 예측이 가능할 것으로 판단하였으며,
전국 단위가 아닌 지역 단위로 분석 범위를 좁히는 것이 모델 성능 개선에 도움이 될 수 있다고 느꼈습니다.
