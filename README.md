# 생성 AI 기반 뉴스 심리지수를 활용한 부동산 가격 예측 모델

## 1. 프로젝트 개요
부동산 가격 예측은 일반적으로 금리, GDP, 물가지수와 같은 거시 경제 지표를 기반으로 수행된다.  
그러나 이러한 지표는 시장 참여자의 심리나 정책 변화에 대한 반응을 충분히 반영하지 못한다는 한계가 있다.

본 프로젝트에서는 **뉴스 기사 감성 분석을 통해 생성한 '뉴스 심리지수'를 부동산 가격 예측 모델에 반영**하여  
기존 시계열 예측 모델 대비 예측 성능 개선 가능성을 분석하였다.

본 연구는 **한국정보처리학회 ACK 학술대회 논문 제출을 목적으로 진행된 프로젝트**이며  
장려상을 수상하였다.

---

## 2. 데이터

### 뉴스 데이터
- 네이버 뉴스 기사 약 **30,600건**
- 기간 : 2019.01.01 ~ 2023.03.31

### 경제 및 부동산 지표
- 아파트 매매가격지수
- 기준금리
- 대출금리
- 명목 GDP
- 전국 아파트 매매 거래량

---

## 3. 데이터 처리 과정

1️⃣ 뉴스 기사 수집  

네이버 뉴스 데이터를 수집하여 월별 뉴스 데이터를 구축하였다.

2️⃣ 뉴스 요약  

뉴스 본문을 요약하기 위해 다음 방법을 사용하였다.

- **TextRank (Gensim)**
- **KoBART**

3️⃣ 감성 분석  

요약된 뉴스 기사에 대해 감성 분석을 수행하여  
뉴스의 감정을 수치화하였다.

- 긍정 : 1  
- 중립 : 0  
- 부정 : -1  

이를 기반으로 **월별 뉴스 심리지수**를 생성하였다.

---

## 4. 모델

본 연구에서는 **Multi-head Attention LSTM 모델**을 사용하여  
부동산 가격을 예측하였다.

모델 설정

- n_heads : 16
- hidden layer : 64 / 32
- optimizer : Adam
- learning rate : 1e-3
- evaluation metric : RMSE

---

## 5. 결과

뉴스 기사 기반 심리지수를 예측 모델에 반영한 결과  

- 기존 모델 대비 **RMSE 감소**
- 향후 **3개월 부동산 가격 예측 성능 향상**

뉴스와 같은 **비정형 텍스트 데이터가 부동산 시장 예측에 유의미한 영향을 줄 수 있음**을 확인하였다.
